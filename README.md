**Resume & Skill Matching Engine**

This project is a data-driven system designed to automate the resume screening and candidate-role matching process. It combines Data Analytics techniques with Machine Learning models to recommend the most suitable internship role for each candidate.

📌 Project Overview

Manual resume screening is time-consuming and inefficient when dealing with a large number of applications. This project solves that problem by building an intelligent system that analyzes resumes and matches candidates to roles based on their skills, experience, and qualifications.

⚙️ Features
Extracts data from resumes (skills, education, tools, experience, certifications)
Converts unstructured resume data into a structured dataset
Implements rule-based matching to calculate skill match percentage
Classifies candidates into Strong Fit, Moderate Fit, and Low Fit
Builds Machine Learning models to predict the best-fit role
Provides Top 3 role recommendations with fit scores
Displays results using a dashboard (Power BI / Streamlit)
🧠 Technologies Used
Python (Pandas, NumPy)
Scikit-learn (ML Models)
Data Preprocessing & Feature Engineering
Excel / Power BI for visualization
📊 Workflow
Collect and extract resume data
Clean and structure the dataset
Define Role–Skill Requirement Matrix
Perform rule-based matching
Apply Machine Learning for role prediction
Visualize results in a dashboard
🎯 Outcome

The system helps reduce manual effort in resume screening and improves the accuracy of role assignment by recommending the best-fit role for each candidate.
